# Commands to Deploy LogTrack to GitHub
# Copy and paste these commands one by one into your terminal

# Step 1: Initialize Git (if not already done)
git init

# Step 2: Add all files
git add .

# Step 3: Create your first commit
git commit -m "Initial commit: LogTrack Phase 1 complete - 6 anomaly detection rules implemented"

# Step 4: Add remote repository
git remote add origin https://github.com/Rishab-Kumar09/LogTrack.git

# Step 5: Set branch to main
git branch -M main

# Step 6: Push to GitHub
git push -u origin main

# Done! Your code is now on GitHub
# Next: Enable GitHub Pages in repo Settings → Pages → Deploy from branch 'main'
# Your site will be live at: https://rishab-kumar09.github.io/LogTrack/
